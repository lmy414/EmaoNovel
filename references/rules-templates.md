# .rules/ 模板

创建 `.rules/` 文件时可参考以下模板。

## 文风规范模板

```markdown
# 文风规范

## 句式
- 短句优先，80% 不超过 15 字
- 少用逗号，用句号断句
- 禁止大段排比

## 节奏
- 动作接动作，不铺过渡
- 每段是一个时刻而非步骤
- 环境和心理描写控制在最小限度

## 对话
- 少用"他说/她说"
- 用身体语言传递情绪
- 人物不道歉不用嘴说——行动代替语言
```

## 输出格式模板

```markdown
# 输出格式

## 章节标题
- 格式：`## 第X章·标题`
- 章节间用 `---` 分隔

## 段落
- 段落间空一行
- 不缩进

## 对话
- 用「」标记对话
- 同一人连续说话合为一段
```

## 禁用词与禁忌模板

