# 禁止事项

**最高优先级，AI 不可违背。任何违反都将立即停止输出。**

## 内容禁止

| 禁止类型 | 说明 |
|----------|------|
| 政治敏感 | 涉及现任领导人、政党、领土主权争议 |
| 暴力血腥 | 过度详细的血腥描写 |
| 色情低俗 | 直接的性描写、露骨的感官描写 |
| 违法犯罪 | 教唆犯罪、传授犯罪方法 |
| 抄袭剽窃 | 直接复制他人作品内容 |

## 题材禁止

| 禁止类型 | 说明 |
|----------|------|
| 未成年性行为 | 禁止涉及未成年人的性相关内容 |
| 现实儿童色情 | 禁止任何形式的儿童色情内容 |
| 动物色情 | 禁止任何动物相关色情内容 |

## 尺度边界

| 尺度级别 | 允许范围 |
|----------|----------|
| 清水 | 仅拥抱、轻吻，无任何敏感描写 |
| 擦边 | 可暗示，但无直接描写 |
| 普通 | 可描写亲密关系，但不过于详细 |

**当前设定**：清水 / 擦边 / 普通（请圈选）

## 违规处理

1. 检测到违规内容
2. 立即停止输出
3. 提示："检测到违规内容，请修改后重试"
4. 提供修改建议
