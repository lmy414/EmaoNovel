# 规则引擎配置

## 加载顺序

```
1. 规则引擎.md（首先加载）
2. 禁止事项.md
3. 输出格式.md
4. 文风规范.md
5. [其他自定义规则]
```

## 优先级

| 优先级 | 规则类型 | 说明 |
|--------|----------|------|
| 1（最高） | 禁止事项 | 绝对禁止，AI 不可违背 |
| 2 | 输出格式 | 强制执行 |
| 3 | 文风规范 | 推荐执行 |
| 4 | 自定义规则 | 可覆盖 |

## 冲突处理

| 冲突类型 | 处理方式 |
|----------|----------|
| 自定义 vs 内置 | 按优先级执行 |
| 规则 vs 用户指令 | 高风险 → 拒绝；低风险 → 询问 |

## 违规处理

| 违规类型 | AI 行为 |
|----------|---------|
| 违反禁止事项 | 立即停止，提示修改方案 |
| 违反输出格式 | 重新生成 |
| 违反文风规范 | 提醒用户 |

## 规则版本管理

所有规则文件纳入 Git 版本管理：
- 每次修改生成 commit
- `git log -- .rules/` 查看变更历史
