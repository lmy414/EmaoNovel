# 输出格式规范

## 章节结构

```
# 第X章 章节标题

[正文...]


```

## 段落格式

- 段落之间空一行
- 每段不超过 200 字

## 对话格式

### 引号格式
```
"对话内容。"角色说。
```

### 破折号格式（可选）
```
——对话内容。
——回应内容。
```

## 场景分割

章节内切换：`---` 分隔符
章节间切换：`### 场景二：XXX`

## 描写标记

| 类型 | 标记 | 示例 |
|------|------|------|
| 心理活动 | [*] | [*]他心想，这次一定要成功。 |
| 旁白 | 无 | 阳光透过窗户洒进房间。 |

## 敏感内容

- 敏感描写用 [敏感内容-简略描述] 代替
