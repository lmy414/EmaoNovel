# 小说创作工作流

基于 Git + GitHub 的轻小说创作版本管理技能。作者自然对话即触发——投喂设定、整理归档、修订迭代、续写存档，全流程由 AI 执行，作者只管"写什么"。

## 触发方式

投喂设定 / 整理项目 / 修订设定 / 续写正文 / 搁置存档——在对话中自然说出意图即可，技能自动激活。

## 目录结构

```
作品名/
├── README.md
├── 作品名-作品定位.md
├── 作品名-世界观设定.md
├── 作品名-角色设定.md
├── 作品名-剧情大纲.md
├── .rules/
├── 原始素材/
├── 正文/
```

## 核心流程

1. 用户触发 → 检查项目文件夹
2. 无 → 初始化完整目录结构 → 读取素材归档
3. 有 → 优先读取 `.rules/` → 按规则执行用户指令
4. 每次修改自动 commit + push，永久可回溯

## 安装

将 `.skill` 文件放入 `~/.openclaw/plugin-skills/` 即可。
