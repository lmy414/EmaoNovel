# 🖋️ EmaoNovel

> 开口就是创作，剩下的交给 AI。

一套以 **Git 为版本管理核心、AI 为执行层** 的小说创作工作流技能。在聊天框里说出想法，AI 完成归档、整理、版本追踪——从零散素材到结构化世界书，从初稿到迭代修订，全自动。

---

## ⚡ 触发方式

### 自然语言（自动匹配）

开口说话，OpenClaw 自动激活技能。无需记忆任何命令：

```
"帮我整理这些设定"
"初始化小说项目"
"把公会改成佣兵体系"
"继续上次写"
"今天先到这"
```

### 精确命令（可选）

需要精确定位某个操作时，用 `/nova` 命令：

| 命令 | 做什么 |
|------|--------|
| `/nova init` | 初始化项目：目录 + 设定模板 + rules + git init |
| `/nova feed` | 读取素材 → 提取信息 → 归类到设定文件 |
| `/nova rev` | 全局搜索 → 列出影响 → 批量修改 → commit |
| `/nova write` | 读取大纲+规则 → 生成正文 |
| `/nova cont` | 读取断点 → 回顾上下文 → 续写 |
| `/nova done` | 标注进度断点 → commit → push |
| `/nova stat` | 当前章节位置 + 最近 commits |
| `/nova rule` | 创建或修改 `.rules/` 规则文件 |

**自然语言和命令可以混用。** 不需要记，想说就说，想用命令就用命令。

## 🔄 执行流程

```
你说一句话
    │
    ▼
检查项目文件夹
    │
 ┌──┴──┐
 有     无
 │      │
 ▼      ▼
读取   问作品名+一句话定位
已有     ↓
项目   创建最小骨架（README + 正文/）
状态     ↓
 │    git init → commit
 │      │
 └──────┘
    │
    ▼
执行你的意图
    │
 ┌──┴──┬────┬────┬────┐
 ▼     ▼     ▼     ▼     ▼
feed  write  rev  cont  done
生成   写    批量   续   存档
设定   正文  修订   写   推送
```

**项目按需生长**——init 只给最小骨架，设定、规则、正文都是后续逐步加进来的。

## 🧠 规则引擎

每个项目自带 `.rules/` 规则目录，换项目 = 换规则 = AI 行为立刻切换：

```
.rules/
├── 总规则.md           # AI行为 + 加载顺序 + 冲突处理 + 禁止事项
│   ├── 禁止词库.md      # 绝对禁用词汇
│   ├── 敏感词库.md      # 需谨慎使用的词汇
│   ├── 自定义规则.md    # 用户扩展入口
│   ├── 检查清单.md      # 关键节点自检
│   └── 规则变更日志.md  # 规则修改记录
├── 版本管理规则.md      # Git策略 + commit规范
├── 结构规则.md          # 目录结构 + 输出格式 + 文风
│   └── 参考文风.md      # 文风参考文本
└── 内容规则.md          # 角色关系、剧情红线、世界观一致性、术语
```

**渐进生长**：init 只给前三个规则，内容规则和自定义规则在创作中按需添加。

## 🔧 高度可扩展

- **换项目** → 换一套 `.rules/`，AI 行为立刻切换
- **加规则** → 新建一个 `.md` 放进去，AI 自动读取
- **定制模板** → `references/templates/` 里十套开箱即用模板
- **工作流扩展** → 规则引擎支持任意自定义规则文件，不限于小说创作

## 📦 项目结构

```
EmaoNovel/
├── README.md                       # 本文件，人类可读的项目文档
├── SKILL.md                        # 技能声明（元数据 + 概述 + 资源指针）
└── references/
    ├── workflow.md                 # 执行流程全集（命令映射、操作细节、规则管理、文风分析）
    ├── rules.md                    # 规则引擎架构（架构、原则、冲突处理）
    ├── usage-examples.md           # 典型使用场景
    └── templates/                  # 全套规则模板
        ├── 总规则.md               #   AI行为准则 + 加载顺序 + 禁止事项
        ├── 版本管理规则.md          #   Git策略 + commit规范（唯一权威）
        ├── 结构规则.md              #   目录结构 + 输出格式 + 文风
        ├── 内容规则.md              #   叙事约束 + 术语统一 + 尺度边界
        ├── 自定义规则.md            #   用户扩展入口
        ├── 禁止词库.md              #   绝对禁用词汇
        ├── 敏感词库.md              #   需谨慎使用的词汇
        ├── 参考文风.md              #   文风参考文本
        ├── 检查清单.md              #   关键节点自检
        └── 规则变更日志.md           #   规则修改记录
```
